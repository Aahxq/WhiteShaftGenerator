# PR 白轴生成器

PR 白轴生成器是 PromeRotation 内置插件，订阅 PR 的 LogSystem 事件并导出 PR Timeline JSON 与 Cactbot timeline 文本。

## 功能

- 记录敌对目标/Boss 读条、敌方技能效果、Boss 增益、队伍成员减益、点名和连线。
- 普通攻击是独立事件类型，默认关闭，可由用户决定是否记录。
- 关闭普通攻击后会自动移除当前列表里的普通攻击，导出时也会再次过滤。
- 默认不记录玩家自己释放的技能、自身增益、队伍输出技能和状态移除事件。
- 地图/对象事件和场地标记可手动开启，用于补充特殊机制线索。
- 对象出现、可选中、不可选中按 `DataId` 去重；同一类型同一 `DataId` 只保留一条，不同 `DataId` 分别保留。
- 战斗结束自动导出默认关闭，可由用户手动开启。
- 仅在战斗中开始记录；战斗结束后保留记录列表，支持勾选条目后只导出选中内容。
- 记录列表和导出备注显示为 `技能名(来源:来源名 目标:目标名) +秒数`，BOSS 读条保留技能 ID 并额外显示读条秒数。
- 支持手动开始、导出、导出选中、删除未选、清空和自定义导出目录。
- 导出的 `.json` 可由 PR Timeline 编辑器继续整理；同名 `.txt` 可复制到 PR 时间轴的 `从Cactbot导入`。

## 安装

构建后将输出目录作为一个子目录放入 PromeRotation 配置目录的 `Plugins` 文件夹，然后在 PR 插件管理器中启用。

## 默认导出目录

未设置自定义目录时，文件会写入:

`PromeRotation配置目录/PureTimelines/WhiteShaftGenerator`

第一次使用或目录不存在时会自动创建 `WhiteShaftGenerator` 子目录。

每次导出会生成同名 `.json` 和 `.txt` 两个文件。
